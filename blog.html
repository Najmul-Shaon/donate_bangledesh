What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, where each node corresponds to a part of the document (such as elements, attributes, or text).

How do you select an element from the DOM?

To select an element from the DOM, you can use methods like getElementById() to select an element by its unique ID, getElementsByClassName() to select elements by their class name, or getElementsByTagName() to select by tag name. More flexible options include querySelector() to select the first element that matches a CSS selector and querySelectorAll() to select all elements that match a given selector. These methods allow you to access and manipulate HTML elements dynamically through JavaScript.

What is event delegation in the context of the DOM, and why is it useful?

1. Performance Optimization: Rather than attaching multiple event listeners to individual child elements, event delegation uses just one listener on the parent. This reduces memory usage and speeds up the handling of dynamic elements.
2. Handles Dynamic Content: If child elements are added or removed dynamically (e.g., items added to a list), event delegation ensures that the parent continues to listen for events on those newly added elements without needing additional event listeners.
3. Cleaner Code: Instead of managing multiple event listeners for potentially many child elements, you manage one listener on a parent, which simplifies your code.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles in the DOM, you can use methods like setAttribute(), getAttribute(), and removeAttribute() to work with attributes such as src, href, and alt. For example, setAttribute("src", "new-image.jpg") changes the image source. For styles, the style property allows direct manipulation of inline CSS by setting properties like backgroundColor or width. Additionally, the classList property helps manage CSS classes—using methods like add(), remove(), and toggle() to dynamically modify an element's appearance by adding or removing predefined styles.
